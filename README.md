# Ditto JS SDK | Webpack Example

*Minimal example demonstrating how to integrate Ditto in a Webpack app.*

## Getting Started

- Run `npm install`
- Run `npm run start`
- Open the shown localhost address in browser
- Edit `app/main.js` and reload your browser
